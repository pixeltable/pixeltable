## Overview

Build and share multimodal AI datasets without managing infrastructure. Work with your images, videos, audio, and documents through a unified Python API - process them with AI models, create embeddings, and publish your results for team collaboration or public research.

## Quick Start

**Replicate a dataset:**
```python
import pixeltable as pxt
pxt.replicate('pxt://pixeltable:fiftyone/coco_mini_2017', 'local_copy')
```

**Publish your datasets** (requires account and API key from [signin.pixeltable.com](https://signin.pixeltable.com/)):
```python
pxt.publish('my_table', 'pxt://username/dataset', access='public')
```

Learn more in the [Data Sharing Guide](https://docs.pixeltable.com/notebooks/feature-guides/data-sharing).
