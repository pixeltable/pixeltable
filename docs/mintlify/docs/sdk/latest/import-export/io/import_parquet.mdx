## Args

<Note>
Parameter optional/required status may not be accurate if docstring doesn't specify defaults.
</Note>

<ParamField path="table" type="str" required>
  Fully qualified name of the table to import the data into.


</ParamField>

<ParamField path="parquet_path" type="str" required>
  Path to an individual Parquet file or directory of Parquet files.


</ParamField>

<ParamField path="schema_overrides" type="Optional[dict[str, Any]]" >
  If specified, then for each (name, type) pair in `schema_overrides`, the column with name `name` will be given type `type`, instead of being inferred from the Parquet dataset. The keys in `schema_overrides` should be the column names of the Parquet dataset (whether or not they are valid Pixeltable identifiers).


</ParamField>

<ParamField path="primary_key" type="str | list[str] | None" >
  The primary key of the table (see \[`create_table()`\]\[pixeltable.create_table\]).


</ParamField>

<ParamField path="kwargs" type="Any" required>
  Additional arguments to pass to `create_table`.


</ParamField>

## Returns

<ResponseField name="return" type="any" required>
  A handle to the newly created table.


</ResponseField>
