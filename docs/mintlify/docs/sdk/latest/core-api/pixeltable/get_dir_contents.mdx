## Args

<Note>
Parameter optional/required status may not be accurate if docstring doesn't specify defaults.
</Note>

<ParamField path="dir_path" type="str" >
  Path to the directory. Defaults to the root directory.


</ParamField>

<ParamField path="recursive" type="bool" >
  If `False`, returns only those tables and directories that are directly contained in specified directory; if `True`, returns all tables and directories that are descendants of the specified directory, recursively.


</ParamField>

## Returns

<ResponseField name="return" type="any" required>
  A \[`DirContents`\]\[pixeltable.DirContents\] object representing the contents of the specified directory.


</ResponseField>

## Raises

### `Error`

If the path does not exist or does not designate a directory.


## Examples

```python
Get contents of top-level directory:

>>> pxt.get_dir_contents()

Get contents of 'dir1':

>>> pxt.get_dir_contents('dir1')
```
