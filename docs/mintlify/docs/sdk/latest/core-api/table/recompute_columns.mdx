## Parameters

<ParamField path="columns" type="any" required>
  The names or references of the computed columns to recompute.

</ParamField>

<ParamField path="errors_only" type="any" >
  If True, only run the recomputation for rows that have errors in the column (ie, the column's `errortype` property indicates that an error occurred). Only allowed for recomputing a single column.

</ParamField>

<ParamField path="cascade" type="any" >
  if True, also update all computed columns that transitively depend on the recomputed columns.

</ParamField>

## Examples

```python
Recompute computed columns `c1` and `c2` for all rows in this table, and everything that transitively
depends on them:

>>> tbl.recompute_columns('c1', 'c2')

Recompute computed column `c1` for all rows in this table, but don't recompute other columns that depend on
it:

>>> tbl.recompute_columns(tbl.c1, tbl.c2, cascade=False)

Recompute column `c1` and its dependents, but only for rows that have errors in it:

>>> tbl.recompute_columns('c1', errors_only=True)
```
