iterated over in row-major order (left-to-right, then top-to-bottom). An optional `overlap` parameter may be specified. If the tiles do not exactly cover the image, then the rightmost and bottommost tiles will be padded with blackspace, so that the output images all have the exact size `tile_size`.


## Constructor

```python
TileIterator(
    self,
    image: PIL.Image.Image,
    *,
    tile_size: tuple[int, int],
    overlap: tuple[int, int] = (0, 0)
)
```
