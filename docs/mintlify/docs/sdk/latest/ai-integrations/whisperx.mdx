### `transcribe()`

Transcribe an audio file using WhisperX.


This UDF runs a transcription model *locally* using the WhisperX library, equivalent to the WhisperX `transcribe` function, as described in the [WhisperX library documentation](https://github.com/m-bain/whisperX).

WhisperX is part of the `pixeltable.ext` package: long-term support in Pixeltable is not guaranteed.

**Requirements:**

- `pip install whisperx`


**Signature:**

```python
transcribe(audio: Audio,
     model: String,
     compute_type: Optional[String],
     language: Optional[String],
     chunk_size: Int) -> Json
```

**Parameters:**

- **`audio`** (*Audio*): The audio file to transcribe.

- **`model`** (*String*): The name of the model to use for transcription.


**Returns:**

- *Any*: A dictionary containing the transcription and various other metadata.



**Example:**

<Tip>

Add a computed column that applies the model `tiny.en` to an existing Pixeltable column `tbl.audio`
of the table `tbl`:
```python
 tbl.add_computed_column(result=transcribe(tbl.audio, model='tiny.en'))
```

</Tip>
