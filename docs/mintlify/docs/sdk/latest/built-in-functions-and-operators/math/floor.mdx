Equivalent to Python [`float(math.floor(self))`](https://docs.python.org/3/library/math.html#math.ceil) if `self` is finite, or `self` itself if `self` is infinite. (This is slightly different from the default behavior of `math.floor(self)`, which always returns an `int` and raises an error if `self` is infinite. The behavior of Pixeltable generalizes the Python operator and is chosen to align with the SQL standard.)


## Signature

```python
pixeltable.functions.math.floor(self: Float) -> Float
```
