These interactive demos showcase how Pixeltable simplifies building AI applications by providing unified data infrastructure for multimodal workflows.

All demos are open source and available on our [Hugging Face Account](https://huggingface.co/Pixeltable). You can duplicate any space to run it locally or deploy your own version.

## Content Generation & Analysis

[Video to Social Media Post Generator](https://huggingface.co/spaces/Pixeltable/video-to-social-media-post-generator)
Transform videos into engaging social media content with AI-powered analysis and generation

[AI Video Analyzer with OpenAI](https://huggingface.co/spaces/Pixeltable/ai-video-analyzer-gpt4-vision-tts-narration)
Generate rich video narratives with multi-style analysis and TTS narration

[Collaborative Story Builder](https://huggingface.co/spaces/Pixeltable/Collaborative-Story-Builder)
Create stories collaboratively with AI assistance and version control
