## Features Highlighted

- **Unified Data Management**: Handle multiple data types through a simple interface
- **Automatic Versioning**: Built-in version control for all transformations
- **Computed Columns**: Express complex operations declaratively
- **Seamless Integration**: Works with popular AI tools out of the box
