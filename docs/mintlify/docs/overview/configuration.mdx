### Configuration Options
Pixeltable can be configured through:
- Environment variables
- System configuration file (`~/.pixeltable/config.toml` on Linux/macOS or `C:\Users\<username>\.pixeltable\config.toml` on Windows)

Example `config.toml`:
```toml
[pixeltable]
file_cache_size_g = 250
time_zone = "America/Los_Angeles"
hide_warnings = true
verbosity = 2

[openai]
api_key = 'my-openai-api-key'

[label_studio]
url = 'http://localhost:8080/'
api_key = 'my-label-studio-api-key'
```

### System Settings

<ResponseField name="PIXELTABLE_HOME" type="string" default="~/.pixeltable">
  Pixeltable user directory
</ResponseField>

<ResponseField name="PIXELTABLE_CONFIG" type="string" default="$PIXELTABLE_HOME/config.toml">
  Pixeltable config file location
</ResponseField>

<ResponseField name="PIXELTABLE_PGDATA" type="string" default="$PIXELTABLE_HOME/pgdata">
  Directory where Pixeltable DB is stored
</ResponseField>

<ResponseField name="PIXELTABLE_DB" type="string" default="pixeltable">
  Pixeltable database name
</ResponseField>

<ResponseField name="PIXELTABLE_FILE_CACHE_SIZE_G" type="float" required>
  Maximum size of the Pixeltable file cache, in GiB
</ResponseField>

<ResponseField name="PIXELTABLE_TIME_ZONE" type="string" default="system time zone">
  Default time zone in [IANA format](https://en.wikipedia.org/wiki/List_of_tz_database_time_zones)
</ResponseField>

<ResponseField name="PIXELTABLE_HIDE_WARNINGS" type="boolean" default={false}>
  Suppress warnings generated by various libraries used by Pixeltable
</ResponseField>

<ResponseField name="PIXELTABLE_VERBOSITY" type="integer" default={1}>
  Verbosity for Pixeltable console logging (0: minimum, 1: normal, 2: maximum)
</ResponseField>
