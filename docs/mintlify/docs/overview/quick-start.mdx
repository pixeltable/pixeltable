# Building Your First AI App

Pixeltable apps work in two phases:
1. Define your app structure (once)
2. Use your app (anytime)

<Steps>
  <Step title="Install Pixeltable">
    ```bash
    pip install pixeltable
    ```
  </Step>

  <Step title="Define Your App Structure" icon="table">
    Create `schema.py`:
    ```python
    import pixeltable as pxt
    from pixeltable.functions import openai

    # Initialize app structure
    pxt.drop_dir("chatbot", force=True)
    pxt.create_dir("chatbot")

    # Define data schema
    conversations = pxt.create_table(
        path_str="chatbot.conversations", 
        schema_or_df={"prompt": pxt.String}, 
        if_exists="ignore"
    )

    # Configure processing pipeline
    conversations.add_computed_column(
        messages=[{"role": "user", "content": conversations.prompt}]
    )

    conversations.add_computed_column(
        response=openai.chat_completions(
            messages=conversations.messages,
            model="gpt-4o-mini",
        )
    )

    conversations.add_computed_column(
        answer=conversations.response.choices[0].message.content
    )
    ```
  </Step>

  <Step title="Use Your App" icon="play">
    Create `app.py`:
    ```python
    import pixeltable as pxt

    # Connect to your app
    conversations = pxt.get_table("chatbot.conversations")

    # Use it!
    conversations.insert([
        {"prompt": "Write a haiku about dogs."}
    ])

    # Get results
    print(conversations.collect())
    ```
  </Step>
</Steps>

## What Makes This Different?

<Steps>
  <Step title="Persistent Storage" icon="database">
    Every interaction is automatically stored and versioned:
    ```python
    # Query past conversations
    conversations = pxt.get_table("chatbot.conversations")
    ```
  </Step>

  <Step title="Type-Safe Pipeline" icon="diagram-project">
    Catch errors before runtime with schema validation:
    ```python
    # Your pipeline is validated when defined
    conversations.add_computed_column(
        messages=[{"role": "user", "content": conversations.prompt}]
    )
    ```
  </Step>  
  
  <Step title="Production Ready" icon="rocket">
    Scale from prototype to production without rewriting:
    ```python
    # Insert many prompts at once
    conversations.insert([
        {"prompt": p} for p in prompts
    ])
    ```
  </Step>
</Steps>
