<Accordion title="Configuration Options">
Pixeltable can be configured through:
- Environment variables
- System configuration file (`$PIXELTABLE_HOME/config.toml`)

Example `config.toml`:
```toml
[pixeltable]
file_cache_size_g = 250

[openai]
api_key = 'my-openai-api-key'

[label_studio]
url = 'http://localhost:8080/'
api_key = 'my-label-studio-api-key'
```
</Accordion>

### System Settings

<ResponseField name="PIXELTABLE_HOME" type="string" default="~/.pixeltable">
  Pixeltable user directory
</ResponseField>

<ResponseField name="PIXELTABLE_CONFIG" type="string" default="$PIXELTABLE_HOME/config.toml">
  Pixeltable config file location
</ResponseField>

<ResponseField name="PIXELTABLE_PGDATA" type="string" default="$PIXELTABLE_HOME/pgdata">
  Directory where Pixeltable DB is stored
</ResponseField>

<ResponseField name="PIXELTABLE_DB" type="string" default="pixeltable">
  Pixeltable database name
</ResponseField>

<ResponseField name="PIXELTABLE_FILE_CACHE_SIZE_G" type="float">
  Maximum size of the Pixeltable file cache, in GiB
</ResponseField>

### API Configuration

<CardGroup cols={2}>
  <Card title="OpenAI">
    ```bash
    OPENAI_API_KEY=sk-...
    ```
    Or in config.toml:
    ```toml
    [openai]
    api_key = 'sk-...'
    ```
  </Card>
  <Card title="Anthropic">
    ```bash
    ANTHROPIC_API_KEY=sk-ant-...
    ```
    Or in config.toml:
    ```toml
    [anthropic]
    api_key = 'sk-ant-...'
    ```
  </Card>
  <Card title="Mistral">
    ```bash
    MISTRAL_API_KEY=...
    ```
    Or in config.toml:
    ```toml
    [mistral]
    api_key = '...'
    ```
  </Card>
  <Card title="Replicate">
    ```bash
    REPLICATE_API_TOKEN=...
    ```
    Or in config.toml:
    ```toml
    [replicate]
    api_token = '...'
    ```
  </Card>
</CardGroup>
