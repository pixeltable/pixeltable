## Quick Example

<CodeGroup>
```python Quick Start
import pixeltable as pxt

# Create a video table
videos = pxt.create_table('videos', {'video': pxt.VideoType()})

# Automatic frame extraction
frames = pxt.create_view('frames', videos, 
    iterator=FrameIterator.create(video=videos.video))

# Add AI processing - only runs on new data
frames.add_computed_column(
    detect_objects=yolox(frames.frame, 
    model_id='yolox_tiny', 
    threshold=0.25)
)
```

```python RAG Example
# Create document table
docs = pxt.create_table('knowledge_base', 
    {'document': pxt.DocumentType()})

# Automatic chunking and embedding
chunks = pxt.create_view('chunks', docs,
    iterator=DocumentSplitter.create(
        document=docs.document,
        limit=300
    ))

chunks.add_embedding_index(
    'text',
    idx_name='minilm_idx',
    string_embed=sentence_transformer.using(
        model_id='sentence-transformers/all-MiniLM-L12-v2'
    )
)
```
</CodeGroup>
